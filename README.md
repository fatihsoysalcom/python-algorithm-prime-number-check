# Python Algorithm Prime Number Check

This example demonstrates a simple algorithm in Python to check if a given number is prime. It illustrates the concept of an algorithm as a sequence of step-by-step instructions to solve a specific problem, a fundamental concept for beginners in programming.

## Language

`python`

## How to Run

Save the code as `main.py`. Open your terminal or command prompt, navigate to the directory where you saved the file, and run it using `python main.py`.
